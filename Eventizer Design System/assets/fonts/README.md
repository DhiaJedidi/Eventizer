# assets/fonts/

Place `Mistrully.ttf` (or `Mistrully.otf`) here to activate the script/accent font.

The `@font-face` rule in `tokens/fonts.css` already points to this path:

```css
src: url('../assets/fonts/Mistrully.ttf') format('truetype');
```

**Steps:**
1. Purchase the commercial license at https://creatypestudio.co/mistrully
2. Upload `Mistrully.ttf` into this folder
3. The font activates immediately — no other edits needed

Until then, `Dancing Script` (Google Fonts) renders as the live fallback.
